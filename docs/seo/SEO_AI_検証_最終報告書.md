## 🚨 100億円達成に向けた最大の課題

### 1位: Product Schema未実装（影響度: **最高**）

**現在の状況**:
- Shopify Storefront API では商品data を完全取得可能
- HTMLには metadata がまったく出力されていない
- Google Shopping に接続不可
- AI検索での引用が低確率

**実装により得られる効果**:
```
└─ Google Merchant Center連携
   ├─ Google Shopping Ads 対応
   ├─ Google Shopping tab（無料）
   ├─ Product Rich Results
   └─ AI検索での引用率 +310%

└─ Amazon/楽天との競争条件の整備
   └─ 同一消費者層への同一検索条件での露出
```

**工数**: 220時間（2-3週間）  
**優先度**: **S（最高、即実装）**

### 2位: llms.txt + GEO対応（影響度: **高**）

**背景**:
- ChatGPT / Google Gemini / Claude / Perplexity の普及
- 2026年: 米国検索の30-35%が AI検索に移行見込み
- 日本でも Q2-Q3から本格化

**Astromeda の課題**:
- /llms.txt ファイル未存在
- robots.txt では AI Bot を Allow しているが、priority指定なし
- 商品description が「スペック」のみで LLM非推奨

**実装により**:
```
├─ llms.txt で優先度指定
│  └─ "/products/" を最優先
│  └─ Astromeda=ブランド認識UP
│
└─ 自然言語テキスト + Cite構造
   └─ ChatGPT が「Astromeda の ONE PIECE PC は...」と引用
   └─ 引用元リンククリック +20-30%
```

**工数**: 128時間  
**優先度**: **S（AI検索の時代到来）**

### 3位: コンテンツマーケティング（影響度: **高**）

**現在の状況**:
- ブログ機能ゼロ
- 教育コンテンツなし（「初心者向けPC選び方」など）
- IP関連ガイド記事なし

├─ ブランド検索（Astromeda）: ~30%
