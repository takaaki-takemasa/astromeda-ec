# コード変更 vs 管理画面で完結する変更 — 使い分けガイド (v1)

このガイドは **「Cowork に何を頼むべきか・自分で admin で済ませるべきか」** の判断基準です。

---

## 大原則

```
🟢 admin で済む変更 → admin で自分で
🟡 コード変更が必要  → Cowork に頼む
```

**理由**: admin で済む変更は **即時反映** (1 分)。コード変更は GitHub Actions が動いて **4-5 分** かかる。デプロイ枠を無駄に使わないことで、本当にコード修正が必要な時のスピードが守られます。

---

## 🟢 admin で自分でやること (Cowork に頼まなくて OK)

### 商品まわり
- 商品の追加・編集・削除
- 商品の価格変更 (※ ただし 02_絶対にやらないこと.md の「お金が動く場所」ルールも遵守)
- 商品の在庫数変更
- 商品の画像差し替え
- 商品にタグを付ける・外す
- 商品を「公開」/「下書き」に切り替え
- 商品の説明文 (リッチテキストエディタで編集)

### コレクションまわり
- コレクションの新規作成 (handle・条件・タグ条件)
- コレクションの並び順変更
- 既存コレクションの説明文・画像変更

### バナー・トップページまわり
- ヒーローバナーの追加・編集・削除
- IPコラボバナー (26マス) の追加・編集・削除
- バナーのリンク先変更 (URL ピッカーでコレクション/記事/外部URL)
- マーキー (流れる文字) の追加・編集

### 記事 (CMS) まわり
- ブログ記事・SEO 記事の追加・編集・削除
- 記事に「関連コレクション」を設定 (patch 0153)

### サイト設定
- 法務情報 (特商法・プライバシー) の編集
- お店情報 (会社名・電話番号・営業時間)
- メニュー (ナビゲーション) の追加・編集

### マーケティング
- キャンペーンの新規作成 (期間・割引)
- セール商品の指定

→ **これらは admin で 1〜3 分で完結**。Cowork に頼まなくて OK。

---

## 🟡 Cowork に頼むこと (コード変更が必要)

### UI 配置・デザイン変更
- トップページの新セクション追加
- 既存セクションの配置入れ替え (商品棚→ABOUT→マーキー の順を変える等)
- 色・フォント・余白の変更
- レスポンシブ (スマホ/PC 表示) の挙動修正

### 新機能追加
- 「レビュー機能を追加」「Q&A 機能を追加」等
- admin 画面に新タブを追加
- 既存機能のロジック変更 (例: ソート順の計算式変更)

### バグ修正
- 「ボタンを押しても何も起きない」
- 「ページが真っ白になる」
- 「特定の条件で価格表示がおかしい」

### admin 設定の構造変更
- 新しい Metaobject 定義の追加 (新フィールド追加)
- 新しい admin タブ追加
- 自動化ルールの変更

→ **これらはソースコード修正が必要**。Cowork に依頼してください。

---

## 判断に迷ったら

**Cowork に聞いてください**:

```
〇〇をしたいですが、admin だけで済むか、コード変更が必要か教えてください。
admin で済むなら、admin の手順を案内してください。
```

Cowork が判断して、最適な経路 (admin or コード) を案内します。

---

## 具体例 — 同じゴールでも経路が違う

### 例 1: 「呪術廻戦」コレクションの説明文を変えたい

✅ **admin で完結**:
1. お店の運営 → 商品・販売 → コレクション
2. 「呪術廻戦」をクリック
3. 「説明」欄を直す → 保存
- 所要時間: 1 分

❌ コード変更が必要なのは:
- 説明欄の **下に新しいリッチコンポーネントを追加** したい場合のみ

### 例 2: 商品「鬼滅の刃 マウスパッド」に「人気」タグを付けたい

✅ **admin で完結**:
1. お店の運営 → 商品・販売 → 商品
2. その商品を編集 → タグ欄に「人気」を追加 → 保存
- 所要時間: 30 秒

❌ コード変更が必要なのは:
- 「人気タグが付いた商品にだけ赤いリボンマークを表示する」など、表示ロジックを変える場合

### 例 3: トップページの 26 マス IP バナーに新IPを追加

✅ **admin で完結** (新IPコラボ出品_完全手順書.html STEP 6 参照):
1. コンテンツ・ページ → 見た目 → IPコラボ → + 新規追加
2. collection_handle / image_url / display_order 入力 → 保存
- 所要時間: 2 分

❌ コード変更が必要なのは:
- 26 マスを 30 マスに増やす場合 (グリッド構造変更)

## まとめチェックリスト

頼む前に自分で確認:

- [ ] admin タブのどこかに、それを編集できる画面はないか?
- [ ] 03_やりたいこと集.md の「🟢 5 分で出来ること」「🟡 10-30 分で出来ること」に該当しないか?
- [ ] 過去に同じことを admin でやった先輩がいないか? (Slack 履歴検索)

YES が 1 つでもあれば、まず admin で試してみる。
NO ばかりだったら、Cowork に頼む。
