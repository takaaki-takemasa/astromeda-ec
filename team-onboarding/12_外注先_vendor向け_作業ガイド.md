# 12. 外注先（vendor）向け 作業ガイド

このページは **デザイン会社・PC 組立委託業者など、Astromeda の社外パートナー** が管理画面を使う時の手順書です。

---

## あなたができること

✅ ゲーミングPCタブ（`/collections/gaming-pc`）のデザインを変える
✅ コラボ以外のゲーミングPC商品（タイトル/写真/価格/在庫）を編集
✅ コラボ以外のコレクション（一覧）を新規作成・編集
✅ ダッシュボード（売上ではなく操作数）を見る
✅ 「🎨 デザイン上書き」タブで HTML/CSS を貼り付けて見た目を一括変更

## あなたができないこと

❌ IP コラボ商品（NARUTO・呪術廻戦・ホロライブ・チェンソーマン等）を触る
❌ トップページ全体のデザイン（ヒーローバナー / IPコラボグリッド等）
❌ 注文・カート・お客様情報・売上データ
❌ メンバー管理・パスワード変更
❌ 法的情報・サイト基本設定
❌ 一括タグ・割引・リダイレクト・お知らせメニューの編集

→ 上記が必要な時は Astromeda 担当者に依頼してください。

---

## 最初にやること

### 1. ログイン
1. Astromeda 担当者から URL とパスワードを受け取る
2. URL を開く → ユーザー名とパスワードを入力 → ログイン
3. 右上に紫色の **「外注先」** バッジが表示されていれば成功

### 2. 触れる範囲を確認

ログイン後、画面左の「お店の運営」をクリックすると、あなたが触れるタブだけが表示されます:
- **📦 商品** (コラボ以外のみ編集可)
- **📦 在庫** (在庫数の調整)
- **📚 コレクション** (一覧の作成/編集)
- **🎨 デザイン上書き** (HTML/CSS でデザイン一括変更)

これ以外のタブは表示されないか、操作してもエラーで止まります（安全のため）。

---

## よくある作業

### A. ゲーミングPCタブのデザインを変える（HTML/CSS）

1. 「🎨 デザイン上書き」タブを開く
2. 「＋ 新規上書きを追加」ボタンを押す
3. **対象セクション**で「ゲーミングPC：◯◯」を選ぶ（ヒーロー / 特集カード / 人気ランキング / パーツで選ぶ / 価格帯で選ぶ / お問い合わせ）
4. **モード**を選ぶ:
   - **元のデザイン** → 何もしない（一旦元に戻す時に使う）
   - **CSS だけ上書き** → 見た目（色・余白・フォント）だけ変える。HTML 構造は元のまま
   - **HTML/CSS を完全に上書き** → そのセクションを丸ごと別の HTML で置き換える
5. HTML / CSS を貼り付けて「保存する」を押す
6. **公開**トグルをオンにすると即座に本番に反映されます

⚠ **安全対策（自動）**:
- `<script>`、`<iframe>`、`onclick` 属性、`javascript:` URL は自動で除去されます
- 上書きでサイトが壊れた場合は「削除」ボタンで元のデザインに戻せます

### B. コラボ以外のゲーミングPCを新規出品

1. 「📦 商品」タブを開く
2. 「＋ 新規追加」を押す
3. 商品名・価格・写真を入力（**注意:** 商品名に「NARUTO」「呪術」「ホロライブ」等の IP 名を入れるとエラーで弾かれます）
4. 商品ジャンル「ゲーミングPC」を選ぶ
5. 標準のプルダウン17項目（CPU/GPU/メモリ等）が自動で付きます
6. 保存 → 即座にサイトに反映

### C. コラボ以外のコレクション（商品の一覧）を作る

1. 「📚 コレクション」タブを開く
2. 「＋ 新規追加」を押す
3. タイトル・URL末尾・説明を入力
4. **注意:** 既存の IP コレクション（`naruto-shippuden`、`jujutsukaisen-collaboration` 等）は編集できません

---

## エラーメッセージが出たら

| エラー | 原因 | 対応 |
|---|---|---|
| 「IP コラボ商品のため、外注先ロールでは編集できません」 | コラボ商品を編集しようとした | Astromeda 担当者に依頼 |
| 「コラボ系タグが付いているため」 | 商品にコラボタグがある | Astromeda 担当者に依頼 |
| 「この操作を実行する権限がありません」 | 触れない機能を使おうとした | このガイドの「できること/できないこと」を再確認 |
| 「ログインしてください」 | セッション切れ | 再ログイン |

---

## 困ったとき

- **画面が真っ白になった** → ブラウザを更新（F5）→ ダメならログアウト → 再ログイン
- **HTML 上書きでサイトが壊れた** → 「🎨 デザイン上書き」タブで該当エントリを「削除」 → 元に戻る
- **何をしていいか分からない** → このガイドの「よくある作業」を見直し、それでも分からなければ Astromeda 担当者に Slack/メールで連絡

---

## 監査ログ

あなたの操作はすべて Astromeda 側で記録されています（誰が・いつ・何を変更したか）。
不正アクセスを疑わせる操作（コラボ商品への編集試行など）は自動で記録され、Astromeda 担当者に通知されます。

これは「監視のため」ではなく「お互いの信頼関係を保つため」「もし不具合が出た時に原因を辿るため」の仕組みです。
