# 10. 用語集 — 知らない言葉が出たらここを引いてください

このシステムで出てくる **専門用語と日常語の対応表** です。
分からない言葉があれば、まずここを引いてください。

## ファイル・倉庫の用語

| 用語 | 意味 | 例 |
|---|---|---|
| **GitHub** | インターネット上のファイル倉庫 | takaaki-takemasa/astromeda-ec |
| **リポジトリ / repo** | 倉庫そのもの | 1 倉庫 = 1 サイト |
| **コミット / commit** | 「ここで一旦保存」のしるし | 写真のスナップショット |
| **プッシュ / push** | 自分の変更を倉庫に送る (= 反映する) | 4-5 分で本物のサイトに反映 |
| **ブランチ / branch** | 倉庫の中の枝分かれ。今は `main` だけ使う | main = 本番に直結 |
| **PR / プルリクエスト** | 「変更をレビューして欲しい」依頼 | 今は使っていない (将来) |
| **マージ / merge** | 枝の変更を main に取り込む | これも将来 |
| **PAT** | Personal Access Token = 個人アクセス鍵 | メンバーが GitHub に書き込むためのパスワードみたいなもの |
| **2FA / 二段階認証** | パスワード + スマホ確認の追加ロック | 全員 ON 推奨 |

## 開発・運用の用語

| 用語 | 意味 | 例 |
|---|---|---|
| **デプロイ / deploy** | 変更を本物のサイトに反映する | push の 4-5 分後に自動で完了 |
| **GitHub Actions** | デプロイを自動で走らせる仕組み | CEO は何もしない・自動 |
| **CI/CD** | デプロイの自動化 = GitHub Actions | 同じ意味 |
| **ビルド / build** | デプロイの一段階。ファイルを準備する作業 | ログに出る |
| **ロールバック / rollback** | 元のバージョンに戻すこと | 「壊れた」時に CEO がする |
| **本番 / production** | お客様が見ている本当のサイト | shop.mining-base.co.jp |
| **ステージング / staging** | テスト用サイト | 今は実質使ってない |

## セキュリティの用語

| 用語 | 意味 | 例 |
|---|---|---|
| **認証 / authentication** | 「あなたは誰?」を確認する | ログイン画面 |
| **認可 / authorization** | 「あなたは何ができる?」を確認 | 役割 (owner/editor 等) |
| **CSRF** | 偽の操作を防ぐ仕組み | システムが裏で守ってる |
| **XSS** | 悪意あるコード注入 | 同上 |
| **branch protection** | main 倉庫を「壊せないようにする鍵」 | CEO が 07 で設定 |
| **force push** | 履歴を消す操作 | branch protection で禁止 |
| **Dependabot** | パッケージの古さを警告する番犬 | GitHub 設定 |
| **Secret scanning** | パスワードを誤って公開してないか見張る | GitHub 設定 |

## 役割の用語 (patch 0156 で導入)

| 用語 | 意味 | できること |
|---|---|---|
| **オーナー / owner** | 一番偉い役割 | 全部できる + メンバー管理 |
| **管理者 / admin** | 2 番目 | 全部できる (メンバー削除以外) |
| **編集者 / editor** | 日常運用 | 商品・コンテンツを編集 |
| **閲覧者 / viewer** | 見るだけ | 修正できない |
| **bootstrap モード** | 個別ユーザーがまだ作られてない初期状態 | ADMIN_PASSWORD 環境変数だけでログイン可 |
| **multi-user モード** | 個別ユーザーで運用 | 1 人作った時点で自動切替 |

## エラーや状態のシグナル

| 表示 | 意味 |
|---|---|
| **🟢 / Success / 200 / 公開済** | 成功・正常 |
| **🔴 / Failed / 400-499** | あなたの操作に問題 (やり直せばOK・データは無事) |
| **🔴 / Failed / 500-599** | システム側に問題 (Cowork に相談) |
| **404 Not Found** | ページが存在しない |
| **403 Forbidden** | 権限がない |
| **401 Unauthorized** | ログインが必要 |
| **CSRF token mismatch** | ページを再読み込みしてやり直し |
| **rate limit exceeded** | 短時間に同じ操作を連発した。少し待つ |

## メンバーが使う頻度の高い 5 語だけ覚える

最低限これだけ:

1. **管理画面 (admin)**: 日常作業の入口
2. **コレクション**: 商品の集まり
3. **デプロイ**: 変更を本物のサイトに反映する
4. **オーナー / 編集者**: 自分の役割を理解
5. **困ったら 09 文書**: トラブルシュート

これ以外は分からなくても運用に支障ありません。
