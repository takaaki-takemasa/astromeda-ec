**v3 (2026/04/23 改訂2) の改善点**:
- **GitHub Desktop のインストール不要** に (調査の結果 CEO も使っていないことが判明)
- Cowork が **Linux サンドボックス内で直接 git を実行** するので、メンバーの PC には何も追加でインストール不要
- メンバーの実質的な技術操作は **「PAT を 1 回だけ発行して貼り付け」だけ** (ブラウザで 5 クリック)
- セットアップ所要時間: **30-40 分** (v1 60-90分 → v2 40-60分 → v3 30-40分)
- git コマンド (pull/commit/push) を **日常語** に隠蔽
- ファイルパス知識不要に (「IP コラボの呪術廻戦」と言えば Cowork が自動で該当箇所を探す)
- **Astromeda 専用スキル** (`.claude/skills/astromeda-team/SKILL.md`) を repo に同梱
  → メンバーが加入した瞬間、Cowork が onboarding を自動誘導

## まず最初にやること (今日)

### CEO がやること

1. このフォルダを Slack/LINE/メール等でメンバーに共有
   ```
   C:\Users\Astromeda\Documents\astromeda-ec_完全版\team-onboarding\
   ```
2. メンバー 4-6 名に **`01_メンバー向け_セットアップ手順.md`** を送る
3. メンバーから GitHub username が返ってきたら **`00_CEO向け_メンバー追加手順.md`** に従って招待

### メンバーがやること

1. **`01_メンバー向け_セットアップ手順.md`** を最初から最後まで実施 (60-90分)
2. **`02_危険操作リスト.md`** を熟読
3. **`03_よくある作業集.md`** をブックマーク
4. 練習作業 (例: 自己紹介を team-onboarding/members/ 配下に追加して push) を 1 回やる
